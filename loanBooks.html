How to Loan Books?

  1. If you have a library card, you are able to request a book online if it's available.
    • * If you don't have a library card, go to a "Register" menu and fill a form to get a library card.
    • * You can check availability of the book at the "Find Books" menu.
  2. Write your library card number and the book's ID below to reserve the book.
  3. After reserving the book, go to the library and pick up the reserved book.
